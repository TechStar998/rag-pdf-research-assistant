# rag-pdf-research-assistant

A Retrieval-Augmented Generation (RAG) application that answers questions from PDF documents using LangChain, FAISS, Hugging Face Embeddings, and Transformer models.

## Features
- PDF document ingestion
- Text chunking
- Vector embeddings
- Semantic search with FAISS
- Context-aware question answering
- Fully free Hugging Face-based pipeline

## Tech Stack
- Python
- LangChain
- FAISS
- Hugging Face Transformers
- Sentence Transformers
